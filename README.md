# SUBSAMPLE #

A set of classes and functions to read a large density field output from the modified 2LPT code DENSITYFIELDS (or any other list of Lmesh x Lmesh x Xmesh (double) numbers in binary format), and subsample accordingly; plus, compute the power spectrum and bispectrum of each subsample.
